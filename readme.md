# GoFrame 聊天室

基于 GoFrame 框架开发的 WebSocket 聊天室应用。

## 项目开发步骤

### 1. 环境准备
- 安装 Go 环境 (1.16+)
- 安装 GoFrame CLI 工具
  ```
  go install github.com/gogf/gf/v2/cmd/gf@latest
  ```

### 2. 项目初始化
- 使用 GoFrame CLI 创建项目
  ```
  gf init chatroom
  ```
- 项目目录结构说明
  - `api`: 对外接口定义
  - `internal`: 内部逻辑代码
  - `resource`: 静态资源文件
  - `manifest`: 配置清单
